PowerVAN 프로그램의 버전별 기능을 설명해 드리겠습니다.

구분 독립 실행형 버전 엔터프라이즈 버전
데이터 별도의 서버 등이 필요 없습니다. 데이터를 공유할 수 있는 서버 컴퓨터가 필요합니다.
데이터 저장소 프로그램을 실행하는 컴퓨터에만 데이터가 저장됩니다. 데이터베이스 서버에 모든 데이터가 저장되므로, 데이터의 공유가 편리합니다.
데이터 백업 이용자가 수동으로 처리해야 합니다. 데이터베이스 서버에 스케줄러를 이용해서 자동으로 가능합니다.
프로그램 다운로드 일반적으로 인터넷에서 다운로드 할 수 있습니다. 별도의 설치 지원이 필요합니다.
이용대상 중/소 규모 기업에 적합 대규모의 기업에 적합
라이브 업데이트 자동
엑셀 수기 업로드 지원 불가 지원 가능
동기화 동시실행 1개 1~10개 가능
(개수가 많을 수록 동기화 속도가 빨라짐)

(별도의 개발 비용이 필요할 수 있습니다.)
프로그램 수정 지원 불가 지원 가능
(별도의 수정 비용이 필요할 수 있습니다.)
